# BakeWithMiki 🍰

BakeWithMiki is a simple multi-page recipe website that allows users to browse dessert recipes such as cakes, cookies, and cake decorations.

## 📌 Project Description
This website was created as part of a software engineering assignment. The main goal is to provide a user-friendly interface for browsing and viewing baking recipes. The website includes multiple pages such as Home, Recipes, Cake Decoration, About, and Contact.

## 🧩 Features
- Multi-page navigation system
- Recipe browsing (cakes, desserts, etc.)
- Individual recipe pages with images and instructions
- Cake Decoration Page 
- About page - Biography section
- Contact page for user interaction
- Clean and simple UI design using HTML and CSS

## 🛠️ Technologies Used
- HTML5
- CSS3

## 📁 Project Structure
index.html
/pages → HTML pages
/css → Styling files
/images → Recipe images


## 🚀 How to Run the Project
1. Download or clone the repository
2. Open `home.html` in any web browser
3. Navigate through the website using the menu
